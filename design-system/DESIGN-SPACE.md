> 杂志编辑 × 暖暗纸墨 · 私人档案馆

## 一、Welcome：封面

Welcome 的本质是**今天的入口**。打开日记本，第一眼看到的是今天的日期。

```
┌──────────────────────────────────────────────────┐
│                                                  │
│                                                  │
│                                                  │
│               2  0  2  6                         │
│               6 月  4 日                          │
│               星期四                              │
│                                                  │
│              下午好。                             │
│                                                  │
│          ┌──────────────┐                        │
│          │  开始写今天   │                        │
│          └──────────────┘                        │
│                                                  │
│              翻翻过去  ↓                          │
│                                                  │
│                                                  │
└──────────────────────────────────────────────────┘
```

**视觉重心：** 今天的日期。大字、宽字距、衬线体。不是数据——是仪式。每一天打开日记库看到当天的日期被这样郑重地呈现，本身就是"你在这里"的锚点。

**层级：**
1. 日期（最大，视觉重心）
2. 时段问候（小字，在按钮上方）
3. 开始写今天（主按钮，唯一亮色元素）
4. 翻翻过去（小字链接，下方）

**细节：**
- 日期字号约 `font-size: 4rem` 年份 + `3rem` 月日 + `1.5rem` 星期，全部衬线体。字距宽松（`letter-spacing: 0.04em`）
- 问候语随系统时间变化：早上好(5-11) / 中午好(11-14) / 下午好(14-18) / 晚上好(18-5)
- 主按钮：暖琥珀底 + 深色文字。圆角 6px。hover 时有微弱暖光扩散
- "翻翻过去"：inline 文字链接，箭头 `↓` 有微弱的上下浮动动画（暗示"往下，往过去"）
- 背景：页面上半部分有极微弱的时间渐变——早晨偏冷蓝灰、下午偏暖黄、晚上偏深蓝。不是明显的色块，只是基底色温的微妙偏移

**为什么不死板：** 日期每天都在变。12 月 31 日和 6 月 4 日打开 Welcome，视觉印象不同。不是静态封面。

## 二、色彩系统

### 2.1 基底（保持现有 `--dv-*`，微调）

| 变量 | 色值 | 用途 |
|------|------|------|
| `--dv-bg` | `oklch(0.135 0.012 275)` | 页面底色（冷调深灰） |
| `--dv-panel` | `oklch(0.18 0.014 275)` | 卡片底色 |
| `--dv-panel-strong` | `oklch(0.205 0.016 275)` | 卡片 hover / 活跃状态 |
| `--dv-border` | `rgba(242, 238, 226, 0.09)` | 分割线、卡片边框 |
| `--dv-text` | `rgba(247, 243, 234, 0.88)` | 正文（暖调白） |
| `--dv-text-muted` | `rgba(226, 230, 226, 0.52)` | 次要文字 |

### 2.2 新增：杂志强调色

| 角色 | 色值 | 用途 |
|------|------|------|
| `--accent` | `oklch(0.68 0.12 60)` | 主按钮、链接、高亮边框（暖琥珀） |
| `--accent-glow` | `oklch(0.68 0.12 60 / 0.2)` | 按钮 hover 发光扩散 |
| `--emotion-cold` | `oklch(0.55 0.08 245)` | 情感·郁 = 冷蓝 |
| `--emotion-warm` | `oklch(0.68 0.1 75)` | 情感·明 = 暖金 |
| `--emotion-hot` | `oklch(0.52 0.12 25)` | 情感·焦 = 暗红 |
| `--ink` | `rgba(247, 243, 234, 0.92)` | 阅读区正文（比 `--dv-text` 更亮） |

### 2.3 纸纹

```css
body {
  /* 极细噪点，打破纯平暗色 */
  background-image: url("data:image/svg+xml,..."); /* 或 CSS noise filter */
}
```

噪点透明度 `0.03-0.04`。不做皮革纹、木纹——只是打破纯平的微纹理。

## 三、排版系统

### 3.1 字体

| 角色 | 字体 | 来源 |
|------|------|------|
| 标题/日期/大号展示 | Noto Serif SC | 已有 |
| 正文阅读 | Noto Serif SC | 已有 |
| UI 辅助/标签/按钮 | Noto Sans SC | 已有 |
| 数据/日期编号/侧栏读数 | Space Mono | Google Fonts 引入 |

### 3.2 字号层级

| 层级 | 大小 | 字体 | 使用场景 |
|------|------|------|----------|
| Display | `4rem` / 64px | Serif | Welcome 日期 |
| H1 | `2.5rem` / 40px | Serif | 月份标题（Timeline section header） |
| H2 | `1.5rem` / 24px | Serif | 日记标题（卡片、Reader） |
| Body | `1rem` / 16px | Serif | 日记正文、卡片导语 |
| UI | `0.875rem` / 14px | Sans | 按钮、侧栏导航 |
| Label | `0.75rem` / 12px | Sans + Mono | 元数据、日期编号、标签 |
| Micro | `0.6875rem` / 11px | Mono | 字数、间距计数、侧栏小字 |

### 3.3 行距与行宽

- 日记正文阅读：`line-height: 1.9`，max-width `36em`（~576px）
- 卡片导语：`line-height: 1.6`，max 3 行
- UI 文字：`line-height: 1.4`

## 四、卡片系统（Timeline）

### 4.1 三种尺寸

| 维度 | 条件 | 布局 | 内容 |
|------|------|------|------|
| Feature（主稿） | 字数 > 2000 或每月最大篇 | 横跨 2 列，宽幅 | 日期 + 标题 + 导语（2-3 行正文） + 情绪色条 |
| Standard（标准） | 字数 500-2000 | 1 列方卡 | 日期 + 标题 + 导语（1 行） + 情绪色条 |
| Compact（紧凑） | 字数 < 500 | 1 列窄卡 | 日期 + 标题 + 情绪色条 |

### 4.2 卡片结构

```
Feature 卡片                          Standard 卡片
┌──────────────────────────────┐    ┌──────────────┐
│ ▌5.9 罗森                     │    │ ▌5.17 抄经    │
│ ▌                            │    │ ▌·明          │
│ ▌这学期的罗森好像总是在下雨。 │    │ ▌             │
│ ▌周围桌子翻了新，还是老样子。 │    │ ▌昨天去了龙华  │
│ ▌我在想一些事情...            │    │ ▌寺...        │
│ ▌              📍图书馆  🖼2  │    │ ▌             │
│ ▌              4287 字       │    └──────────────┘
└──────────────────────────────┘

Compact 卡片
┌──────────┐
│ ▌5.24     │
│ ▌我要生活 │
│ ▌的激情   │
└──────────┘
```

- 左边缘的 `▌` 是**情绪色条**——3px 宽，冷蓝/暖金/暗红/灰色（无标注）
- Feature 卡片比 Standard 宽两倍，在 grid 中产生自然的不对称
- 图中和地点用图标展示，hover 时才显示文字
- 字数只在 Feature 卡片上展示

### 4.3 卡片动效

- hover：卡片背景从 `--dv-panel` 过渡到 `--dv-panel-strong`，200ms
- 情绪色条 hover 时微微增宽到 5px，并增加发光
- 不弹跳、不位移

## 五、图形元素系统

### 5.1 情绪色条

替代文字标签（"郁""明""焦"）。3px 宽的竖线在每张卡片左边缘。颜色映射：

| 情绪 | 颜色 |
|------|------|
| 郁 | `--emotion-cold` 冷蓝 |
| 明 | `--emotion-warm` 暖金 |
| 焦 | `--emotion-hot` 暗红 |
| 无标注 | `--dv-text-dim` 暗灰 |

不需要在页面上出现"郁/明/焦"的文字——色条已经传达了情绪。

### 5.2 图标替代文字

| 信息 | 现在 | 改为 |
|------|------|------|
| 地点 | "图书馆" pill 文字 | 📍 pin 图标，hover 显示地名 |
| 图片数 | "3 张图片" 文字 | 🖼 + 数字（始终显示） |
| 标签 | 文字 pill | 不显示（卡片已有情绪色条） |

### 5.3 月份密度指示

月份标题旁不显示 "9 篇" badge。改为一行小圆点——9 篇 = 9 个点排列：

```
2025 · MAY   ○○○○○○○○○
```

每个点直径 4px，间距 3px。有日记的日子 = 亮色点，无日记的日子 = 暗色点。一眼看出写作频率。

### 5.4 日期编号

卡片上的日期使用 Space Mono 等宽字体 + 小字号。如 `5.9`、`5.17`。统一格式 `M.D`（一位或两位月+日，无前导零，无年份）。年份在 section header 已经显示。

## 六、动效系统

### 6.1 原则

- 阅读区（Reader、卡片正文）不做动画
- 导航/筛选/卡片区做微交互
- 所有动画 200-400ms，带 damping

### 6.2 具体动效

| 位置 | 效果 |
|------|------|
| Welcome 日期 | 无动画（静态仪式感） |
| Welcome 问候语 | 微弱的 fade-in（300ms），每次打开只播放一次 |
| Welcome `↓` 箭头 | 上下浮动 4px，2s 循环 |
| 主按钮 hover | 暖光扩散（box-shadow 增大），200ms |
| 维度筛选条切换 | 选中项加亮+下划线滑入，300ms |
| 卡片 hover | 背景变亮+情绪色条加宽，200ms |
| 月份 section 进入 | 淡入+轻微上移（和现有 page transition 保持一致） |
| 页面切换 | 0.25s opacity + translateY(-4px)，比现在的 0.18s 稍慢 |

## 七、侧栏

```
┌──────────────┐
│              │
│  日 记 库     │  ← 衬线体小字 masthead
│              │
│              │
│  ○ 时间线    │  ← 主导航，活跃态=圆点亮起
│  ○ 录入      │
│  ○ 导出      │
│              │
│              │
│              │
│ ─────────── │
│ 设置 · AI    │  ← 底部小字，分隔线隔开
└──────────────┘
```

- 宽度：180px（从现在的 240px 缩减）
- 导航项：圆点指示器（4px → hover 6px），活跃态圆点变亮并增大
- "录入"可以用更亮的圆点颜色（琥珀），暗示"写"的动作
- "设置"和"AI"不用图标，纯文字链接，极简

## 八、逐页规范

### 8.1 Welcome

- 视觉重心：今天的日期（大字衬线，垂直居中偏上）
- 问候语在日期下方，小字 sans-serif，`--dv-text-muted`
- 主按钮：暖琥珀底色 + 深色字 + 6px 圆角 + hover 发光
- 次级链接："翻翻过去 ↓"，`--dv-text-muted`，hover 变亮
- 背景有极微弱的时间色温渐变

### 8.2 Timeline

- 顶部 inline 筛选栏：横向 pill 条——"全部"、"2024"、"2025"、"2026" + 搜索框 + 情绪色点筛选（蓝/金/红小圆点，点击切换）
- 按月分 section。Section header = 大衬线月份 + 密度点阵
- 卡片：按篇幅自动分配 Feature/Standard/Compact 三种尺寸
- Feature 卡片在其所属月份中充当"主稿"
- Grid：大屏 3 列、Feature 跨 2 列 → 实际布局是流动的
- 搜索/筛选时自动切换到紧凑列表视图（一行一篇）

### 8.3 Reader

保持现有结构，微调：
- 顶部工具栏收窄高度
- 返回按钮改用 `<` 箭头 + 小字"时间线"
- 日期+标题居中，衬线体
- 标签行在底部：情绪色点 + 地点图标 + 图片计数的图标版本
- 四角暗角叠加（CSS radial-gradient 模拟面罩暗角效果）

### 8.4 Editor

- 工具栏统一 `--dv-*` 配色
- 字控按钮从现在的零散按钮改为 grouped 分段控件
- 预设编辑器保持在右侧面板
- 预览区底色从 `#1a1a1f` 改为 `var(--dv-bg)`

### 8.5 Export

保持现有功能，外观改为杂志附录风格：
- 列表项 = 一行一篇，左边勾选框、日期、标题
- 选中态：左边色条 + 背景增亮
