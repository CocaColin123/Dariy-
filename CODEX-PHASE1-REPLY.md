## 视觉风格参考图（确定 Outer Wilds 美术方向）

以下三张参考图在 `D:\电脑管家迁移文件\微信聊天记录搬家\xwechat_files\wxid_pch92215q6dy22_11d9\temp\RWTemp\2026-06\9e20f478899dc29eb19741386f9343c8\` 目录下。

### 图 1：`0212f20aab3cc9e128601b3cd47cb9df.png`

Brandon Grotesque Black 字体 + Outer Wilds 游戏对话场景。

**你从中得到的视觉信息：**
- 对话/标签字体风格：无衬线、粗体、现代感、高易读性。游戏内用白色无衬线文字显示对话，下方有带箭头的对话选项。
- 场景色调：暖棕建筑+冷蓝角色皮肤+绿色植被+暖橙灯光。色彩低饱和度，冷暖对比柔和。卡通透染风格。
- UI 元素：简洁的对话选择界面——白色文字+箭头指示器，没有花哨的装饰。

### 图 2：`3603d01c6b9bc9795fd75a1eea147b19.png`

ITC Serif Gothic 字体 + Outer Wilds 游戏设置界面。

**你从中得到的视觉信息：**
- 标题/菜单字体风格：正式、复古衬线体，兼具"科技感"与"叙事感"。适合日志、菜单、剧情文本。
- UI 配色：深棕/黑色背景 + **橙色（选项卡标题、按钮边框）** + 米白（文字、选中选项卡背景）。这是整个项目最重要的配色参考——暖橙琥珀是 Outer Wilds UI 的灵魂。
- 界面布局：选项卡式设计（GAMEPLAY/TEXT/AUDIO/INPUT/GRAPHICS），功能分类清晰。子选项+开关式设置。

### 图 3：`b8a61de06c02e1aaa0d254a5d5872bf2.png`

Space Mono 字体 + 太空探索 HUD 演示界面。

**你从中得到的视觉信息：**
- HUD/功能文本字体风格：等宽无衬线，复古未来主义。字形简洁机械，带"复古科技感"。**Space Mono 在 Google Fonts 上免费可直接引入。**
- HUD 配色：深灰/黑色背景 + 绿色（功能性状态显示）+ 橙色（高亮交互/进度）+ 白色（操作提示/数值）。
- HUD 元素：圆形燃料/生命体征表、修复进度、重力系数、距离/速度显示、十字准星、操作提示文字。

### 三张图总结的设计语言

| 层级 | 字体 | 颜色 | 用途 |
|------|------|------|------|
| 标题/菜单 | ITC Serif Gothic | 橙/米白 | 主标题、日志条目 |
| 对话/标签 | Brandon Grotesque Black | 白/奶油 | 示波器标签、按钮文字 |
| HUD/数据 | Space Mono | 绿/白/橙 | 数值读数、按键提示、侧栏数据 |

**配色核心：** 暖深棕底 + 橙琥珀强调 + 冷绿荧光状态 + 米白正文。不是纯黑底，不是亮蓝强调。
